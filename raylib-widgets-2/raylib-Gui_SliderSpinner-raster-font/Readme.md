# Gui Slider Spinner

Поєднаний віджет спінера та слайдера з кнопками інкременту/декременту для бібліотеки raylib.

***

## Опис

Цей віджет поєднує функціонал слайдера і двох кнопок зі стрілками для точного регулювання числового значення. Підтримує типи `int` і `float`, а також горизонтальну і вертикальну орієнтації.

Основні можливості:

- Обробка діапазонів зі “зворотньою” логікою (min може бути більшим за max).
- Автоповтор з затримкою і прискоренням при тривалому утриманні кнопок.
- Зміна значення коліщатком миші.
- Малювання стрілок, кнопок і слайдера з ручкою.
- Гнучка кастомізація кольору, шрифту, розмірів і кроку.

***

## Функції

### `bool Gui_SliderSpinner(...)`

Основна функція віджету.

#### Аргументи:

- `int id` — унікальний ідентифікатор для віджета.
- `int centerX, centerY` — координати центру віджета.
- `int width, height` — розміри віджета.
- `const char* textLeft, textRight` — тексти підписів над кнопками або слайдером (можуть бути NULL).
- `void* value` — вказівник на значення (`int` або `float`), яке редагується.
- `void* minValue, maxValue` — межі значення.
- `float step` — крок змін.
- `GuiSpinnerValueType valueType` — тип значення (`GUI_SPINNER_INT` або `GUI_SPINNER_FLOAT`).
- `GuiSpinnerOrientation orientation` — орієнтація віджета (горизонтальна/вертикальна).
- `Color baseColor` — базовий колір віджета.
- `RasterFont font` — шрифт для текстів.
- `int spacing` — відстань між символами в текстах.
- `bool showButtons` — показувати кнопки інкременту/декременту чи ні.

#### Повертає:

- `true`, якщо значення змінилося.
- `false`, якщо ні або аргументи некоректні.

## Приклад використання

```c
int value = 5;
int min = 0;
int max = 10;

bool changed = Gui_SliderSpinner(0, 200, 100, 300, 40,
                                 "Min", "Max",
                                 &value, &min, &max,
                                 1.0f, GUI_SPINNER_INT,
                                 GUI_SPINNER_HORIZONTAL,
                                 (Color){100, 150, 200, 255},
                                 myFont, 2, true);

if (changed) {
    printf("Значення змінилося: %d\n", value);
}
```
